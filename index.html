Backbone List

Please modify ./public/javascripts/lib/backbone.list.js to include the following functionality:

  • Clicking add creates a list item.
  • Clicking remove deletes a list item.
  • Clicking reset restores the list to it's original state.
  • The list only communicates with the UI through it's collection and models.
  • Items remain sorted by text through any operation.

There is also a test suite of other useful functionality that might be fun to implement.
